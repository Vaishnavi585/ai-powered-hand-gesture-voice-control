# AI-Powered Hand Gesture and Voice Control System

## Overview

The AI-Powered Hand Gesture and Voice Control System is a touchless human-computer interaction project that enables users to control system functions and mobile operations using hand gestures and voice commands. The project leverages Computer Vision, Artificial Intelligence, and Speech Recognition technologies to create an intuitive and accessible user experience.

## Features

- Real-time hand gesture recognition
- Voice command integration
- Touchless system control
- Mobile device interaction support
- Webcam-based gesture detection
- Volume and media control
- User-friendly interface
- Fast and accurate gesture tracking

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- PyAutoGUI
- SpeechRecognition
- Pycaw

## File Descriptions

### gestures_updated.py
Core module for real-time hand gesture detection and system control.

### gestures_on_mobile.py
Implements gesture-based mobile control functionality.

### voice_updated.py
Processes voice commands and executes corresponding actions.

### webcam_test.py
Used to verify webcam functionality and video capture.

### Batch_17_mini_project_report.pdf
Contains complete project documentation and implementation details.

## Workflow

1. Capture video from webcam.
2. Detect hand landmarks using MediaPipe.
3. Recognize predefined gestures.
4. Map gestures to system actions.
5. Process voice commands simultaneously.
6. Execute the requested operation.
